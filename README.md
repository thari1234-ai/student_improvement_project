# Student Academic Improvement Project

This project predicts student academic improvement rate using **Polynomial Regression**.

## Features
- Predicts improvement rate of a student over weeks
- Categorizes improvement: High / Moderate / Low
- Shows reasons for improvement or low performance
- Visualizes improvement curve with colored background

## How to Run
1. Clone the repo:git clone https://github.com/thari1234-ai/student_improvement_project.git
cd student_improvement_project
2. Install required libraries:pip install numpy pandas scikit-learn matplotlib
3. Run the project:python main.py
4. Enter **Name** and **Roll Number** when prompted.

## Dataset
- Located in `data/student_data.csv`
- You can add more students or weeks to the dataset
